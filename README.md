# Storyteller-AutoBarrageForDouyuTV
基于https://github.com/Ironieser/Storyteller_Of_Auto-Barrage 制作的全新悬浮窗版自动弹幕发射装置

使用方法：打开直播间页面后复制v3.0.js文件代码至浏览器控制台(F12 console) 你的页面右侧就会出现弹幕发射控制台啦！
